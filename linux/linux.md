# Linux
## Linux를 사용하는 이유
* 대부분의 소프트웨어가 NT가 아닌 Unix를 기반으로 만들어졌다.
* 따라서 Linux의 조상격인 Unix의 기능을 익혀두는 것이 개발 과정에 필요하다.
